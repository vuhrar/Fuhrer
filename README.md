# Führar - النظام القانوني الذكي للقانون العمالي السعودي

[![Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_red.svg)](https://streamlit.io)
[![Python](https://img.shields.io/badge/Python-3.10%2B-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Führar** هو تطبيق ويب متكامل **مجاني 100%** متخصص في **القانون العمالي السعودي**، مبني باستخدام **Streamlit** و **AI**.

## 🚀 المميزات الرئيسية

### 📊 تحليل قانوني متقدم
- **18 محوراً قانونياً** لتغطية جميع جوانب القضية
- تحليل شامل مع **إجماع نهائي**
- دعم للقانون العمالي السعودي كاملاً

### 🔧 أدوات قانونية متقدمة
- **تصنيف آلي** للقضايا القانونية
- **محرك بحث قانوني** في نصوص القانون
- **توليد تفنيد قانوني** آلي
- **حاسبة استحقاقات** متقدمة (نهاية الخدمة، الإجازات)
- **كشف الزلات القانونية** في المراسلات
- **تقييم قوة القضية** من 1-10
- **متابعة المواعيد القانونية** الهامة

### 📄 توليد مستندات
- توليد **دعاوى قانونية** بصيغة DOCX
- توليد **دفاع قانوني**
- توليد **اتفاقيات تسوية**


### 💾 تخزين سحابي
- **Supabase** (قاعدة بيانات سحابية مجانية)
- تخزين **الجلسات** و **القضايا**
- **Backup آلي**

### 🌐 واجهة مستخدم
- **واجهة عربية كاملة** (RTL)
- **ثيم داكن** احترافي
- **متجاوب** مع جميع الأجهزة
